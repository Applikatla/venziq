# VENZIQ brand assets

The mark is a Q whose tail starts inside the bowl, crosses the lower-right edge,
and rises into a verification tick. One shape carries the letter and the proof.

## Files

- `venziq-mark.svg` — standalone mark, single colour (accent lime). Outlined geometry, production-ready.
- `venziq-mark-gradient.svg` — standalone mark in the brand gradient (lime to violet).
- `venziq-lockup.svg` — wordmark + mark reference. Uses the Space Grotesk display font; the
  canonical, theme-aware lockup is the `<Logo>` component in `src/components/Logo.tsx`.
- `../favicon.svg` — compact mark on a dark tile (legible at 16px, stands out in light and dark chrome).
- `../favicon.ico`, `../favicon-16.png`, `../favicon-32.png`, `../apple-touch-icon.png`,
  `../icon-192.png`, `../icon-512.png` — generated favicon set.

## Colour

- Accent (lime, dark mode): `#b6f25f` — proof / verified / live.
- Accent (moss, light mode): `#2f6b0e`.
- Ink (dark mode): `#ecede8`; (light mode): `#0e1116`.

In the lockup the wordmark and Q body are ink; only the tip of the tick carries the accent.
